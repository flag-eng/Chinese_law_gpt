基于华为云 MaaS 的法律智能问答系统设计与实现


在法律咨询领域，普通用户常面临“条文晦涩难懂”与“专业咨询成本高”的双重痛点，通用大语言模型（LLM）虽然具备强大的自然语言处理能力，但在处理严谨的法律问题时，往往存在“幻觉”风险，即编造不存在的法条或进行错误的逻辑推演。
本项目旨在构建一个垂直领域的检索增强生成（RAG）系统。该系统不依赖模型单纯的内部参数记忆，而是通过引入外部法律知识库，利用向量检索与语义重排技术，将用户问题映射到具体的法律依据上，再由大模型基于事实进行回答。这不仅保留了模型的语义理解能力，更通过“挂载知识库”的方式实现了回答的准确性和可追溯性。
